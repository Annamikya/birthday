## 📞 Quick Help

**Images not showing?**
→ Check file path in code matches actual file location
→ Verify image format is .jpg, .png, or .webp
→ Check file is in correct folder: `public/images/`

**Music not playing?**
→ Verify file at: `public/music/background-music.mp3`
→ Try clicking the music player button
→ Check browser console for errors

**Styles look weird?**
→ Clear browser cache (Ctrl+Shift+Delete)
→ Hard refresh page (Ctrl+F5)
→ Restart dev server (Ctrl+C then npm run dev)

**Website won't start?**
→ Run: `npm install`
→ Check Node.js is installed: `node --version`
→ Check npm is installed: `npm --version`
